# Creator Lounge Sync API Between PlayGroundX and PGX Casino

## Final Flow

PlayGroundX Creator creates lounge
→ PlayGroundX saves lounge
→ PlayGroundX sends lounge to PGX Casino API
→ PGX Casino displays lounge under Creator Lounges scroll section
→ User clicks View All to open full Creator Lounge Directory
→ User clicks a lounge card
→ PGX Casino opens Creator Lounge Detail Page
→ User clicks Join Creator Lounge
→ User is redirected to PlayGroundX Creator Room
→ PlayGroundX opens PGX Casino iframe on the left and Creator Panel on the right

# 1. PlayGroundX Creates Lounge

## PlayGroundX Route

POST /api/casino/lounges/create

```ts
import express from "express";
import crypto from "crypto";

const router = express.Router();

router.post("/api/casino/lounges/create", async (req, res) => {
  try {
    const {
      creatorId,
      creatorName,
      tableName,
      gameType,
      casinoGameId,
      minBet,
      maxBet,
      startTime,
      durationMinutes,
      coverImageUrl,
      creatorAvatarUrl,
      description,
      vipOnly,
    } = req.body;

    const loungeId = `pgx_lounge_${crypto.randomUUID()}`;

    const lounge = {
      loungeId,
      creatorId,
      creatorName,
      tableName,
      gameType,
      casinoGameId,
      minBet,
      maxBet,
      startTime,
      durationMinutes,
      coverImageUrl,
      creatorAvatarUrl,
      description,
      status: "scheduled",
      vipOnly: !!vipOnly,
      playgroundxRoomUrl: `https://playgroundx.vip/casino/lounge/${loungeId}`,
      iframeGameUrl: `https://pgxcasino.com/game/${casinoGameId}`,
    };

    // 1. Save lounge to PlayGroundX database
    // await db.creatorLounges.create({ data: lounge });

    // 2. Push lounge to PGX Casino
    const pgxResponse = await fetch(
      "https://pgxcasino.com/api/external/creator-lounges",
      {
        method: "POST",
        headers: {
          "Content-Type": "application/json",
          "X-PGX-API-Key": process.env.PGX_CASINO_API_KEY!,
        },
        body: JSON.stringify(lounge),
      }
    );

    if (!pgxResponse.ok) {
      throw new Error("Failed to sync lounge to PGX Casino");
    }

    res.json({
      success: true,
      lounge,
    });
  } catch (error) {
    console.error("Create lounge error:", error);

    res.status(500).json({
      success: false,
      message: "Could not create creator lounge",
    });
  }
});

export default router;
```

# 5. PGX Casino View All Creator Lounges

## API

GET https://pgxcasino.com/api/creator-lounges

```ts
router.get("/api/creator-lounges", async (req, res) => {
  const {
    gameType,
    status,
    vipOnly,
    search,
  } = req.query;

  const lounges = await db.creatorLounges.findMany({
    where: {
      ...(gameType && { gameType }),
      ...(status && { status }),
      ...(vipOnly && { vipOnly: vipOnly === "true" }),
      ...(search && {
        OR: [
          { tableName: { contains: String(search), mode: "insensitive" } },
          { creatorName: { contains: String(search), mode: "insensitive" } },
        ],
      }),
    },
    orderBy: {
      startTime: "asc",
    },
  });

  res.json({
    success: true,
    lounges,
  });
});
```

# 10. Important Rule

Do not send users directly from PGX Casino lounge cards to PlayGroundX.

Wrong:

```tsx
window.location.href = lounge.playgroundxRoomUrl;
```

Correct:

```tsx
window.location.href = `/creator-lounges/${lounge.loungeId}`;
```

Then on the Creator Lounge Detail Page:

```tsx
window.location.href = lounge.playgroundxRoomUrl;
```
